# SMSmart Text Messaging Documentation

##### 1) Yelp  
Programs: 
- search (default)

Parameters:
- key (required)
- longlat
- near
- category

Sample Text Messages / Responses:  
```
@Yelp search:  longlat: true near: 40.74503998,-73.99879607 category: Pizza key: z
```
```
@Yelp: near: Durham, NC category: bars key: z
```
